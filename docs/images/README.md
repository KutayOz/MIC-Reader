# Screenshots

Bu klasöre aşağıdaki görselleri ekleyin:

| Dosya | Açıklama | Boyut |
|-------|----------|-------|
| `app_icon.png` | Uygulama ikonu | 120x120 px |
| `screenshot_home.png` | Ana ekran (hasta adı girişi + yeni analiz butonu) | 1080x1920 px |
| `screenshot_camera.png` | Kamera ekranı (yatay, plaka görünümü) | 1920x1080 px |
| `screenshot_results.png` | Analiz sonuçları (96-well grid + MIC tablosu) | 1080x1920 px |
| `screenshot_history.png` | Geçmiş ekranı (kayıtlı analizler listesi) | 1080x1920 px |

## Screenshot Alma İpuçları

1. **Gerçek cihazda** screenshot alın (emülatör değil)
2. **Karanlık mod kapalı** olsun
3. **Örnek veri** ile çekin (boş ekranlar değil)
4. Kamera ekranı için **gerçek plaka** kullanın
5. Gerekirse **gizli bilgileri** (hasta adı vb.) sansürleyin

## Boyutlandırma

GitHub README'de `width="200"` ile gösteriliyor. Yüksek çözünürlüklü (1080p) screenshot'lar ideal.
